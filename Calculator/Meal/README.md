# 基本介紹

上面是程式的自解檔

1.所以要下載exe檔的人先安裝7-zip壓縮程式

2.程式是用Python開發 故也請先安裝Python

要下載的直接點入 meal.exe檔即可下載安裝

雙擊meal.py後即可使用

若要欣賞程式碼的話可以點入Detail資料夾(但沒有註解。。。)

這是給不知道晚餐要吃什麼由電腦幫你決定的程式

table.py是讓你換餐廳的列表，預設是花蓮壽豐志學村的餐廳

右鍵編輯table.py在各函數的table列表內可以換成你要的餐廳!

    A).morthing代表早餐
  
    B).evening 代表午餐
  
    C).night   代表晚餐
  
    D).dim     代表宵夜
  
    E).drink   代表飲料
